# **App Name**: StockPilot

## Core Features:

- Product Tracking: Track product quantity and location details within the system.
- Expiration Alerts: Proactively send alerts for items nearing their expiration dates to minimize waste.
- Demand Forecasting: Use machine learning to forecast future demand to aid in stock optimization. Show details using an informative dashboard
- Reorder Suggestions: Provide intelligent reorder suggestions based on demand forecasts and current stock levels. The tool uses LLMs to identify current level drivers in making those suggestions
- Stock Level Overview: A unified dashboard showing real-time stock levels, recent changes, and overall inventory status.
- Inventory Management (CRUD): Enable general Create, Read, Update, and Delete operations for managing inventory items.

## Style Guidelines:

- Primary color: Professional blue (#2E9AFE) to convey trust and stability.
- Background color: Light, desaturated blue (#E0F7FA) for a clean and calming interface.
- Accent color: Muted green (#81C784) to highlight growth opportunities and positive stock movements.
- Font pairing: 'Inter' (sans-serif) for body text and 'Space Grotesk' (sans-serif) for headers.
- Crisp and minimalist icons to represent different product categories and inventory actions.
- Clean and structured layout with clear hierarchy for easy navigation and data comprehension.
- Subtle transitions and loading animations to provide feedback and a polished user experience.
